2. [cite_start]Forráskód és Struktúra [cite: 33-40]

A projekt forráskódja moduláris felépítésű, de a könnyebb hordozhatóság érdekében a H5 fázisban egyetlen futtatható fájlban (main.py) egyesítettük az osztályokat.

Projektstruktúra

ClearView_Project/
├── main.py            # A teljes alkalmazás kódja
├── theater.db         # Az automatikusan generált adatbázis
├── README.txt         # Futtatási útmutató
└── docs/              # HTML dokumentáció (H1-H5)

Osztályok Dokumentációja

  • DatabaseManager: Singleton-jellegű osztály az SQL műveletekhez (CREATE, INSERT, UPDATE).
  • TheaterController: Tartalmazza a calculate_price(height) és check_blocking() metódusokat.
  • TheaterApp: A Tkinter ablak (tk.Tk) és a Widget-ek kezelője.

Futtatási Útmutató

A program futtatásához Python 3.10+ szükséges.

python main.py
