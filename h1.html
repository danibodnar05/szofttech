3. Programozási nyelvek és eszközök

  • Python 3.10+: Gyors fejlesztés, gazdag könyvtárkészlet.
  • SQLite: Szervermentes, hordozható adatbázis.
  • PyCharm / VS Code: Modern IDE hibakeresővel.
  • Git & GitHub: Verziókezelés és kollaboráció.
  • Tkinter / PyQt: Asztali GUI keretrendszer.

4. Tervezett módszerek

  • Agile (Scrum-szerű): 2 hetes sprintek, folyamatos visszacsatolás.
  • Tesztelés: Unit tesztek (logika) és Manuális tesztek (GUI).
  • Telepítés: `PyInstaller` segítségével .exe fájl készítése.
